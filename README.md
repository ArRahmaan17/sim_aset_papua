## Installation

1. Clone the repository:

   ```bash 
    git clone https://github.com/ArRahmaan17/sim_aset_papua.git
    ```

2. Install All Dependencies

    ```bash
    composer install
    ```
3. Setup Env

    ```bash
    cp .env.example .env
    ```
3. Generate Application Key

    ```bash
    php artisan key:generate
    ```
## Usage

1. Start development server 

    ```bash
    php artisan serve
    ```
